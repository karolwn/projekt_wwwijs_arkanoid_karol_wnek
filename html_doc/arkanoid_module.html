arkanoid_module module¶

This is a simple Arkanoid clone

arkanoid.py is the main script that keeps everything running (game) required modules:

  • pygame, developed on version 1.9.6

  • ball_module.py

  • block_module.py

  • player_module.py

  • powerup_module.py

  • powerup_handler_module.py

For more complete game with menus and level editor launch main_menu_module.py

Created as a project for the WWW and Script Languages classes at AGH University of Science and Technology Karol Wnęk ICT studies, second year, may 2020
